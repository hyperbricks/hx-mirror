Welcome to the Responsive Menu Demo

This demo showcases how htmx-mirror can be used to dynamically load a navigation menu once via HTMX and automatically mirror it into both mobile and desktop views — without duplicate requests or JavaScript.

  • On mobile, click the ☰ hamburger icon to open the menu.
  • On desktop, the menu appears as a sidebar on the left.
  • Resize the window to switch between mobile and desktop views.

The menu content is loaded only once from menu-fragment.html and mirrored to both views using the hx-mirror extension. This keeps your UI consistent and efficient.
